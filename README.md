# ngx-dynamicComponent

> 项目基于angular2 (版本号 7.0.2)开发,使用ng-cli构建完成，主要实现:

1. 动态组件创建
2. 组件属性绑定:样式属性，赋值，事件
3. 拖拽组件
4. 预览



## 下载

```
  git clone https://github.com/bojue/ngx-DynamicComponent.git

  cd ngx-DynamicComponent

  npm install
```

## 启动

>  npm run start

## 构建

> ng build

## 更新历史

1. 组件动态创建，更新，删除
2. 属性绑定
3. 拖拽，拖拽边界问题优化
